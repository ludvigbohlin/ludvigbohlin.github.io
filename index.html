ShimmerCat creates fast websites with AI, automation and HTTP/2 technology


ShimmerCat is simply added on top of your current e-commerce website where it takes the place of a conventional edge server.

No need for any changes to your current setup.

ShimmerCat handles the communication with the website visitor while also collecting valuable data that is sent to an AI engine and analysis service, and optimization rules are sent back.

Automatic optimizations using statistical inference and other machine learning methods.

The optimizarion rules are used to deliver a faster loading website to the visitor, using features like:

An image of a bolt

HTTP/2 Push

Optimize the delivery of web resources to browsers, by automatically leveraging HTTP/2 Push. It uses the time during which the backend is processing a request to send in advance assets needed by that request to the visitor.


More Info

An image of checkmark

Bot Blocking

Bots can account for a lot of traffic in the site. By having machine-learning separate good from bad bots, and de-prioritize traffic from certain bots, ShimmerCat can increase security and allocate more computing resources to human visitors.


More Info

An image of a file export

Strategic Asset Delivery

By collecting and analysing visitor data, ShimmerCat can set asset priorities for e-commerce websites with focus on your website visitors, so that they can start reading and interacting with your site right away.
